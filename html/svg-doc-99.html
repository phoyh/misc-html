Einfuehrung

SVG wurde und wird von der W3C durch die SVG Working Group entworfen. Es befindet sich noch in der Anfangsphase. Die bisherigen Festlegungen koennen noch wegen Implementierungsfragen geaendert werden. Daher wird an dieser Stelle nur auf die wichtigsten Aspekte von SVG eingegangen, da diese wohl schon jetzt feststehen.

Bisher wurden zwei Entwuerfe am 11.2.99 und 12.4.99 veroeffentlicht. Die MIME-Type wurde auf image/svg festgelegt. Eine SVG wird in XML eingebunden und ist eine fast beliebige Kombination von 2D-Vektorgraphik, Images und Text. Events und Links koennen mit Teilen der Graphik verbunden werden. Verschiedene Anwendungen von SVGs im Zusammenhang mit Animationen ist denkbar.

Strukturtags

Die SVG ist in das XML-Dokument eingebettet. Nach dem Tag fuer den Beginn <svg> koennen zunaechst Graphikelemente innerhalb vom <defs> Tag definiert werden. Dies ist dann noetig, wenn auf diese Elemente innerhalb der SVG zugegriffen wird. Daher sind auch id-Kennummern vonnoeten. Ausserdem koennen hier auch globale Einstellungen an der SVG vorgenommen werden (mit <style> und <script>). Das <desc> Tag folgt optional nach dem </defs>. Zum Schluss kommen noch die eigentlichen Graphikdaten. Graphikelemente koennen mit dem <g> Tag gebuendelt werden, wenn sie Gemeinsamkeiten haben.

Strukturtags innerhalb der SVG Beispiel oder Erlaeuterung
svg <svg width="2in" height="4in">
<!-- eigentliche SVG-Daten -->
</svg>
desc <desc>
<!-- Hier steht die Beschreibung -->
</desc>
g <g style="fillcolor:green" id="Gruppierung 1">
<!-- Elemente der Gruppierung -->
</g>
defs <defs>
<-- Elemente mit id -->
</defs>
style Muss innerhalb des defs-Tags stehen, gilt global fuer die SVG, sonst wie bei HTML
script Muss innerhalb des defs-Tags stehen, gilt global fuer die SVG, sonst wie bei HTML


Graphikelemente

Die eigentlichen Graphikelemente koennen beliebig innerhalb des Koordinatensystems angeordnet werden. Das Fuellen von Flaechen kann mit Mustern oder Gradienten erfolgen.

Mit dem <path> Tag koennen alle anderen 2D-Graphikelemente gezeichnet werden. Ein Pfad besteht aus mehreren Eckpunkten, die mit Linien oder Kurven verbunden sind. Damit koennen alle denkbaren 2D-Graphikobjekte geschaffen werden. Ausserdem kann ein Pfad auch die Form eines Textzuges bestimmen.

Bilder werden aehnlich wie bei HTML mit image (bei HTML: img) eingebunden. Zusatzattribute sind x und y. Ausserdem muss die source mit href (da als XLink) angegeben werden.

Graphiktags innerhalb des SVG-Dokumentes Beispiel oder Erlaeuterung
text <text style="font-size: 12"> Schreibt mit 12px-Font </text>
rect <rect x="100" y="100" width="200" height="40" />
circle <circle cx="30" cy="30" r="10" onMouseOver="invokeProc()"/>
path <path d="M 30 30 L 60 30 L 45 45 z" />
ellipse <ellipse major="40" minor="20" style="fill:blue;" />
polyline <polyline verts="40,40 50,40 50,50 40,50" />
polygon
image <image x="100" y="100" style="width: 100px; height: 100px" href="Bild.png">


Darstellung der SVG

Die SVG wird innerhalb eines gewissen rechteckigen Bereichs (sog. Viewport) dargestellt. Dieser Viewport kann die Graphik in verschiedenen Koordinatensystemen darstellen. D.h., es kann z.B. gezoomt und rotiert werden. Weiterhin koennen Masks definiert werden (z.B. als PNG), um zu definieren, welche Bereiche einer Gruppierung angezeigt werden sollen. Blending-Techniken und Optionen fuer das Clipping von Teilen der Graphiken werden auch unterstuetzt.


Beispiel

Da die momentanen Browser noch nicht fuer XML, geschweige denn fuer das noch experimentelle SVG ausgestattet sind, muss an dieser Stelle die Darstellung von Code reichen.
Es handelt sich hierbei um ein Kreissegment, auf dessen Rand ein Textzug geschrieben ist, und in dessen Inneren ein Logo ist.

<?xml version="1.0" standalone="yes"?>
<svg width="4in" height="3in" xmlns='http://www.w3.org/Graphics/SVG/svg-19990412.dtd'>
 <desc>
  Dies ist ein Beispiel
 </desc>
 <g>
  <path id="Kreis" style="visibility: hidden" d="D 100 100 0 0 80 80 0" />
  <text><textpath href="#Kreis"/>
   Dies ist der Text auf dem Kreis
  </text>
  <circle cx="100" cy="100" r="90">
  <circle cx="100" cy="100" r="70">
 </g>
 <image x="60" y="60" style="width: 80; height: 80" href="Logo.png">
</svg>
